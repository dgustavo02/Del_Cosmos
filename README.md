# DEL COSMOS

**DEL COSMOS** es un juego de disparos en 2D hecho en Python usando la libreria Pygame, donde controlas una nave espacial que debe esquivar y destruir meteoritos que caen del espacio.

---

## Controles

- **Flecha izquierda**: Mover nave hacia la izquierda  
- **Flecha derecha**: Mover nave hacia la derecha  
- **Espacio**: Disparar láser  
- **Enter**: Empieza el juego en el menu principal
- **Esc**: Salir del juego

---

## Requisitos

- Python 3.7 o superior  
- Pygame

---
## Cómo ejecutar
1. Ejecuta tu terminal o cmd en Windows en la carpeta donde esta el archivo main.py
2. Escribe "python main.py" y presiona enter para ejecutar

## Cómo jugar

1. Al iniciar el juego, aparecerá un menú principal.
2. Presiona **ENTER** para empezar a jugar o **ESC** para salir.  
3. La nave tiene una barra de salud que disminuye al chocar con meteoritos.  
4. Dispara a los meteoritos para eliminarlos antes de que lleguen a la nave.  
5. El juego termina cuando la salud de la nave llega a 0.  
6. Al perder, puedes presionar **ENTER** para volver al menú principal y reiniciar la partida.
